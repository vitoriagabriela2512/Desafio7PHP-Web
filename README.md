# 📚 Projeto POO em PHP – Exercícios 1 a 11

## 📌 Descrição Geral

Este projeto reúne uma série de exercícios práticos desenvolvidos em **PHP com Programação Orientada a Objetos (POO)**.

Cada exercício foi estruturado com:
- Interface em HTML (formulário)
- Classe PHP separada
- Lógica de cálculo e retorno de dados
- Estilização com CSS

O objetivo é aplicar conceitos fundamentais de POO de forma prática e progressiva.

---

## 🧠 Conceitos Trabalhados

- Encapsulamento
- Classes e Objetos
- Métodos com e sem retorno
- Construtores
- Estruturas condicionais (`if`, `switch`)
- Operações matemáticas
- Organização em arquivos
- Interação com formulários (POST)


Cada pasta contém:


exercicioXX/
│
├── index.php # Formulário + execução
├── Classe.php # Lógica orientada a objetos
└── README.md # Explicação do exercício
